GDAO: 그래프 기반 Multi-Agent 개발자 커리어 지원 에이전트

비교적 소형 LLM들의 Multi-Agent 협업 시스템으로 개발자의 커리어 성장을 지원하는 챗봇 연구. 각 에이전트가 정보 분석, 경력 추천, 결과 검증 등 독립적인 역할을 수행하며 협력하는 오케스트레이션 아키텍처를 설계 및 구현.

연구 목적

개발자의 커리어 성장을 효과적으로 지원하기 위해, 비교적 소형 LLM들로 구성된 Multi-Agent Orchestration 기반의 챗봇 시스템을 설계·구현하는 것을 목표로 함. 단일 대형 모델에 의존하지 않고, 각기 전문화된 역할을 수행하는 복수의 에이전트가 협력하여 고품질의 커리어 조언을 생성하는 새로운 아키텍처를 제안.

연구 내용 및 방법

정보 분석, 커리어 추천, 결과 검증 등 서로 다른 역할을 담당하는 다수의 소형 LLM 에이전트를 설계하고, 에이전트 간 협업 흐름을 제어하는 오케스트레이션 파이프라인을 구축. LangGraph 기반 Workflow를 통해 에이전트들이 사용자의 커리어 관련 질의를 단계적으로 분석하고 응답을 생성하도록 구현. 사용자 프로파일 정보를 기반으로 맞춤형 피드백을 제공하는 메커니즘을 포함.

Multi-Agent Orchestration 기반 개발자 커리어 지원 시스템 구조

연구 결과

소형 LLM들의 역할 분담 협업을 통해 단일 대형 모델에 준하는 커리어 관련 응답 품질을 달성. 오케스트레이션 구조를 통한 에이전트 간 협력이 응답의 일관성과 정확성 향상에 기여함을 확인. 현재 Expert Systems With Applications (ESWA, SCIE 등재지)에 투고하여 심사 진행 중.

활용 가능 방안

개발자 취업·이직 지원 플랫폼, IT 기업의 기술 스택 기반 채용 매칭 시스템, 개인화된 학습 경로 추천 서비스 등에 적용 가능. 소형 모델 활용으로 운영 비용을 절감하면서도 실용적인 수준의 서비스 품질을 유지할 수 있어 스타트업·중소기업 환경에서도 경제적으로 도입 가능.

멀티모달 유해 밈 탐지를 위한 Self-Attention 기반 이미지-텍스트 통합 모델

SNS에서 확산되는 유해 밈의 이미지-텍스트 복합 특성을 포착하기 위해 Self-Attention 기반 멀티모달 통합 모델을 제안. 이미지 인코더와 텍스트 인코더의 출력을 교차 어텐션으로 융합하여 단일 모달 대비 탐지 성능을 유의미하게 향상시켰으며, KCC 2025에서 일반부문 우수논문상을 수상.

연구 목적

SNS에서 빠르게 확산되는 유해 밈(Hateful Meme)은 이미지와 텍스트가 결합된 복잡한 형태를 띠어, 단일 모달 접근법으로는 탐지에 근본적 한계가 존재. 이미지와 텍스트 정보를 효과적으로 통합하여 유해 밈을 자동으로 탐지하는 고성능 멀티모달 AI 모델을 개발하는 것을 목표로 함.

연구 내용 및 방법

Vision Transformer(ViT) 기반 이미지 인코더와 BERT/RoBERTa 기반 텍스트 인코더를 각각 적용하여 두 모달의 특징을 독립적으로 추출. Self-Attention 메커니즘 기반의 교차 어텐션(Cross-Attention) 모듈로 이미지와 텍스트 표현을 동적으로 융합하는 파이프라인 설계. 표준 Hateful Memes Dataset을 활용하여 학습 및 평가 수행, 다양한 단일·멀티모달 베이스라인과 성능 비교.

Self-Attention 기반 이미지-텍스트 통합 멀티모달 탐지 파이프라인

연구 결과

제안된 Self-Attention 기반 이미지-텍스트 통합 모델이 단일 모달(이미지 전용·텍스트 전용) 및 단순 결합(Concatenation) 방식 대비 유해 밈 탐지 성능(AUROC, Accuracy)에서 유의미한 향상을 달성. KCC 2025 한국컴퓨터종합학술대회에서 일반부문 우수논문상 수상.

활용 가능 방안

카모플라주 치료 시스템을 위한 Sliding Window-LSTM 모델 기반 배터리 잔존 수명 예측

카모플라주 치료 시스템의 안정적 운용을 위해 Sliding Window 기반 LSTM 모델로 배터리 잔존 수명(RUL)을 예측하는 방법론을 제안. 윈도우 크기 조정을 통해 학습 데이터의 다양성을 확보하고 LSTM의 장기 의존성 포착 성능을 극대화. KCI 등재 저널에 게재.

연구 목적

카모플라주(Camouflage) 치료 시스템은 피부 치료용 의료 광원 장치로, 지속적이고 안정적인 전력 공급이 운용의 핵심 전제조건. 배터리 잔존 수명(Remaining Useful Life, RUL)을 정확하게 예측하여 시스템의 안전한 운용을 보장하고, 배터리 교체 시점을 최적화함으로써 불필요한 비용과 서비스 중단을 최소화하는 것을 목표로 함.

연구 내용 및 방법

Sliding Window 알고리즘을 적용하여 배터리 충·방전 시계열 데이터에서 다양한 크기의 학습 샘플을 생성, 학습 데이터의 양과 다양성을 동시에 확보. LSTM(Long Short-Term Memory) 모델로 배터리 시계열의 장기 의존성(long-range dependency)을 포착하여 RUL을 예측. 다양한 윈도우 크기에 따른 모델 성능 변화를 실험적으로 분석하여 최적 설정을 도출하고, GRU·Vanilla RNN 등 비교 모델과의 성능 비교 실험을 수행.

Sliding Window 기반 학습 데이터 생성 및 LSTM RUL 예측 모델 구조

연구 결과

제안된 Sliding Window-LSTM 방법론이 기존 단순 LSTM, GRU 모델 대비 배터리 RUL 예측에서 MAE, RMSE 지표 기준 우수한 성능을 달성. Sliding Window를 통한 데이터 증강 효과가 모델의 일반화 성능 향상에 유의미하게 기여함을 실험적으로 입증. 본 연구는 한국정보기술학회논문지(KIIT)에 게재되어 KCI에 등재됨.

활용 가능 방안

의료 장비의 배터리 수명주기 예측 및 교체 최적화 시스템, IoT 기반 웨어러블 디바이스의 스마트 배터리 관리, 전기차(EV)의 배터리 교체 시점 예측 및 관리, 산업용 로봇·드론의 배터리 모니터링 등 배터리를 사용하는 다양한 디바이스의 예지보전(Predictive Maintenance)에 폭넓게 적용 가능.

멀티모달 데이터 기반 식물 피해 미세 진단

식물 병해충 피해를 조기에 탐지하기 위해 이미지 데이터(CNN)와 온습도·조도 등 환경 센서 시계열 데이터(Transformer)를 결합하는 CNN-Transformer 하이브리드 구조를 제안. 단일 모달 모델 대비 미세 진단 정확도를 향상시켜 스마트팜 환경에서의 적용 가능성을 입증. KSC 2022 학부생부문 장려상 수상.

연구 목적

스마트팜 환경에서 식물의 병해충 피해를 조기에 정밀 탐지하기 위해, 시각적 이미지 정보와 환경 센서 시계열 데이터를 함께 활용하는 멀티모달 AI 진단 시스템을 개발하는 것을 목표로 함. 육안 또는 단일 이미지 분석의 한계를 극복하여 초기 단계의 미세한 피해 증상까지 조기에 감지함으로써 작물 손실을 최소화하고자 함.

연구 내용 및 방법

식물 이미지 데이터는 CNN(Convolutional Neural Network)으로 공간적 외형 특징을 추출하고, 온도·습도·조도 등 환경 센서의 시계열 데이터는 Transformer의 Self-Attention으로 시간적 패턴을 분석. 두 모달의 특징을 효과적으로 결합하는 CNN-Transformer 하이브리드 아키텍처를 설계 및 제안. 병해충 피해 식물 이미지와 동기화된 환경 데이터셋으로 학습·평가하며, 단일 모달 기반 모델들과 성능 비교 실험을 수행.

이미지(CNN) + 환경 센서 시계열(Transformer) 결합 멀티모달 진단 모델 구조

연구 결과

제안된 CNN-Transformer 하이브리드 모델이 단일 이미지 기반 CNN 모델 대비 식물 피해 미세 진단 정확도에서 향상된 성능을 달성. 환경 센서 데이터를 추가 입력으로 활용함으로써 시각적으로 유사한 피해 유형 간 구분 능력이 향상됨을 실험적으로 입증. KSC 2022 한국소프트웨어종합학술대회 학부생부문 장려상 수상.

활용 가능 방안

스마트팜 자동화 관리 시스템, 농업용 IoT 플랫폼 기반 작물 건강 모니터링, 식물원·온실 통합 관리 솔루션, 정밀 농업(Precision Agriculture) 기반 병해충 조기 경보 시스템, 농업 드론과 결합한 광역 포전 모니터링 서비스 등에 적용 가능.

상관관계 기반 특징 개선 기법을 활용한 환자 건강 예측 모델

환자 건강 데이터에서 상관관계 분석을 통해 핵심 특징을 선별하고 모델 복잡도를 줄이면서도 예측 정확도를 유지하는 방법론을 제안. 불필요한 변수를 제거해 학습 효율을 높이고, 임상 환경에서의 빠른 진단 및 의사결정 지원 가능성을 입증. KSC 2022 발표.

연구 목적

상관관계 행렬 분석 기반 핵심 특징 선택 파이프라인 및 분류 성능 비교

HAI(하이)! — Hecto AI Challenge

Hecto 그룹의 채용 연계 AI 경진대회로, 우수 성과자를 인사팀 채용 풀에 직접 등록하는 방식으로 운영. 팀 내 모델 설계 및 데이터 전처리 파이프라인 구축을 90% 이상 담당하여 Hecto 인사팀 HR 등록 성과를 달성. 최종 순위 25/748 (Top 3.3%)

핵심 도전 과제

평가지표가 Log Loss인 대회 특성상, 모델이 특정 클래스에 과신뢰(Overconfidence)할 경우 점수가 급격히 하락하는 문제가 있었음. 또한 실내/외, 낮/밤 등 촬영 환경이 다양해 도메인 편차(Domain Shift)가 발생하였으며, 학습에 방해가 되는 노이즈 이미지 처리가 핵심 과제였음.

접근 전략

노이즈 이미지를 제거하고, Optuna 기반 증강 조합 탐색과 5개 모델 앙상블 + TTA로 과신뢰 문제를 해소하는 방향으로 전략을 수립했습니다.

  • Noise 이미지 제거 — 학습을 방해하는 이상 각도 샘플 사전 제거로 검증 Log Loss 개선
  • Optuna 증강 탐색 — 다양한 촬영 환경 대응을 위해 상위 K개 증강 파이프라인 혼합 적용
  • 2단계 실험 전략 — 클래스당 약 20샘플의 소규모 서브셋으로 빠르게 탐색 후, 유망 설정만 전체 데이터로 학습
  • 5개 모델 앙상블 + TTA — ConvNextV2, EffNetV2, MaxVit, Regnety, SeResNext 조합으로 예측값 평활화
  • 소프트 앙상블 가중치 — 검증 Log Loss 기반 가중치를 소프트닝하여 특정 모델 과의존 방지

성과

748팀 중 최종 25위 달성 (Top 3.3%). Hecto 인사팀 HR 채용 풀에 직접 등록되는 성과를 달성했으며, 팀 내 모델 설계 및 파이프라인 구축의 90%를 직접 담당.

DL용 기법인 Knowledge Distillation을 ML 모델에 맞게 변형 적용하는 방식으로 문제를 해결했습니다.

  • Teacher Model — 54개 피처 전체로 학습 데이터를 분류, 각 샘플의 클래스별 확률값 리스트 저장
  • Temperature Softening — Knowledge Distillation 논문의 Temperature Sigmoid 함수를 적용하여 확률값을 소프트닝 후 pickle 저장
  • Student Model — 테스트 데이터의 19개 피처만 사용, Teacher의 소프트 확률 아웃풋을 타겟으로 학습
  • 분류 처리 — Student Model의 회귀 예측값에 임계값을 적용해 정상/이상 오일로 최종 이진 분류
